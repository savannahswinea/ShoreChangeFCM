# ShoreChangeFCM

These scripts and datasets are replication materials for "Mental models of coastal futures inform responsive management", written by SH Swinea, AR Hughes, MJ Osland, CC Shepard, KB Thorne, JB Alemu I, R Bardou, and SB Scyphers.

EdgeNodeList.xlsx contains individual-level mental model data. FCM_Concept_PresAbs.xlsx contains presence/absence information for the concepts in individual mental models.

Aggregation_ShoreChangeFCM.py is a Python script to aggregate mental model data elicited through online survey platforms like Qualtrics. Sensitivity_ShoreChangeFCM.py is a Python script to run sensitivity analyses to understand how changing concepts in models impacts other concepts.

Sensitivity_MangMarsh_FullRange.xlsx and Sensitivity_SeawallMar_FullRange.xlsx contain the full range of results for the sensitivity analyses we ran.

ShoreChangeFCM.R is an R script to replicate Figs. 4 and 5 (local vs. technical knowledge of mangrove expansion and shoreline hardening).
